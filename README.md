
# QuizzSite-backend
### Startup:
1. Create  file `.env` with parameters:
   - APP_HOST
   - APP_PORT
   - DB_HOST
   - DB_PORT
   - DB_NAME
   - DB_USER
   - DB_PASSWORD
   - REDIS_HOST
   - REDIS_PORT
   - SECRET_AUTH_KEY
   - ALGORITHM
   - EXPIRE_DELTA_TOKEN_MINUTES
   - SMTP_USER
   - SMTP_PASSWORD
2. Run docker:
   ```docker-compose up -d```
